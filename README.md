# 🧑‍💼 Online Employee Management System

A web-based HR management system that helps organizations manage employee records, departments, attendance, and more — all from one place.

---

## ✨ Features

- 🔐 Admin login & role-based access
- ➕ Add, edit, delete employee records
- 🏢 Department & role management
- 📅 Attendance & leave tracking
- 🔍 Search & filter employees

---

## 🛠️ Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | HTML, CSS, JavaScript |
| Backend | Java, JSP |
| Database | MySQL |
| Server | Apache Tomcat |
| IDE | Eclipse |

---

## 🚀 Getting Started

### Prerequisites
- Java JDK 8+
- Eclipse IDE
- Apache Tomcat
- MySQL Server

### Installation

```bash
git clone https://github.com/Ashinsanie535/Online_Employee_Management_System.git
```

1. Open Eclipse → `File > Import > Existing Projects into Workspace`
2. Select the cloned folder
3. Create a MySQL database and import the SQL file from `/full project`
4. Update DB credentials in the config file
5. Deploy on Tomcat and visit:

```
http://localhost:8080/Online_Employee_Management_System
```
